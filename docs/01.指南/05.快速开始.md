## 使用

根据官方的要求，你需要在 `.vitepress/theme/index.ts` 文件中引入 teeker 主题。如果没有该路径，你需要先创建它。

```typescript
import Teeker from "vitepress-theme-teeker";
import "vitepress-theme-teeker/index.css";

export default {
  extends: Teeker,
};

```
