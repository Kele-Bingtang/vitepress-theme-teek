## 使用

根据 Vitepress 的要求，需要在 `.vitepress/theme/index.ts` 文件中引入 Teeker 主题。如果没有该路径，需要先创建它。

```ts
import Teeker from "vitepress-theme-teeker";
import "vitepress-theme-teeker/index.css";

export default {
  extends: Teeker,
};
```
