## 使用

项目拉取

```bash
git clone https://github.com/Kele-Bingtang/vitepress-theme-kt.git
```

依赖安装（只能用 pnpm 安装依赖）

```bash
pnpm install
```

引用包构建

```bash
pnpm stub
```

项目启动

```bash
# 基本文档 Demo
pnpm docs:base:dev
# 多语言文档 Demo（默认语言的文档放到根目录下）
pnpm docs:locales:dev
# 多语言文档 Demo（默认语言的文档放到指定目录下）
pnpm docs:localesRoot:dev
```
